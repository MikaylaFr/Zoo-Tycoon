# Zoo-Tycoon
Project 2 for CS 162
Text based Zoo Tycoon game
Developed in Vim and tested in Unix enviroment.

Animals that can be purchased:
Penguin
  Cost: $1000
  Number of Babies: 5
  Base Food Cost: 10
  Payoff: Cost * 0.01
Tiger
  Cost: $10,000
  Number of Babies: 1
  Base Food Cost: 50
  Payoff: Cost * 0.02
Turtle
  Cost: $100
  Number of Babies: 10
  Base Food Cost: 5
  Payoff: Cost * 0.05
  
UserDefined w/ user defined costs and attributes
